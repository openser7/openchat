# NodejsChat
1. Instalar Node.js LTS (Estable) (NPM)
2. Clonar el repositorio.
3. Moverte a esa carpeta : c:\\ o unix / 
4. Ejecutar : npm install  (Para generar los paquetes de express, socket.io, async)
5. Ejecutar : node /server/server.js
